Abstract

We will be working with Catherine Kuhn who studies changes in Alaskan lakes from a rapidly changing climate as part of the NASA ABoVE project. We will be using data collected for several Alaskan lakes via both field campaigns monitoring lake chemistry and satellite imagery of lake color from Landsat. In Alaska alone, there are over 24,000 lakes so the big issue Catherine would like us to resolve is showing the large-scale structure of this dataset while also allowing a user to drill-down to a specific lake of interest. We aim to do this by building an interactive map of Alaska showing the lake locations, from which a user can then select a specific lake. Once a lake is selected, a pop-out will appear containing a gif of Landsat imagery showing the evolution of the lake’s color over time, time series plots of the satellite’s color channels, and other chemical data for the lakes if available. Given the large size of the Landsat images, we will create these gifs and time-series plots in Google Earth Engine, export them, and then build our interactive map dashboard using D3. Our visualization will be for a technical audience from this field.
